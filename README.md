# ArgentBank

Projet réalisé dans le cadre de la formation Développeur Web OpenClassrooms.

## Aperçu

ArgentBank est une application bancaire permettant à un utilisateur de se connecter à son espace personnel, consulter ses informations et modifier son profil.

Ce projet m'a permis de travailler avec React, Redux Toolkit, React Router et la consommation d'une API REST sécurisée par authentification JWT.

## Objectifs du projet

- Mettre en place une application React moderne
- Gérer l'authentification utilisateur
- Utiliser Redux Toolkit pour le state management
- Consommer une API REST
- Protéger les routes privées
- Modifier les informations d'un utilisateur connecté
- Documenter une API avec Swagger

## Installation

### Cloner le projet

```bash
git clone https://github.com/Lilya3/ArgentBank.git
```

### Front-End

```bash
cd Frontend
npm install
npm run dev
```

### Back-End

```bash
cd Backend
npm install
npm run dev:server
```

## API Documentation

Swagger disponible sur :

```text
http://localhost:3001/api-docs
```

## Démo

Ce projet nécessite un back-end Node.js et une base de données MongoDB pour fonctionner.
